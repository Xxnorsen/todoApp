# Todo List App

A simple and clean To-Do List mobile app built with React Native and Expo.

<img width="200" height="500" alt="image" src="https://github.com/user-attachments/assets/e7e06391-055d-46a1-b930-02a5cbd2fe72" />

## Features

* Add, edit, and delete tasks
* Mark tasks as complete
* Simple login screen
* Clean and minimal design

## Tech Stack

* React Native (Expo)
* JavaScript / TypeScript
* Redux Toolkit (for state management)

## Run the App

```bash
npm install
npm start
npx expo start
```
